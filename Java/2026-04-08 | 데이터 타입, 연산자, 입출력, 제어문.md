### `+` 의미

- java: 산술, 결합

        ```java
        System.out.println("100"+100); // 100100
        System.out.println(100+"100"); // 100100
        ```

## Operation

### 전치, 후치 증가

- 전치 증가
    - `++i`
- 후치 증가
    - `i++`

### 자바 정수 연산의 기본은 int (4byte) 공간에서 계산

```java
byte b = 100;
byte b2 = 1;

// byte b3 = b + b2; 컴파일러는 int + int >> int
// byte + short 도 컴파일러가 int + int 로 연산
// int 보다 작은 타입 다 int로 바꾸어서 연산
// long은 예외
```

- char + **int** → int
- int + int → int
- int + **long** → long
- 정수 + **실수 →** 타입 크기에 상관 없이 실수가 승자

### 삼항연산자

```java
int res = (p==k) ? p : f;
```

### &, &&, |, ||

- 연산자 비트 연산
    - `|`, `&`
    - 영상 처리, 데이터 판독에서 자주 사용
- 논리 연산
    - `&&`, `||` (short circuit)
    - 모든 조건을 다 볼 필요가 없다 → 성능 개선 가능
        - ex1) && 연산 시 거짓인게 있다 → 뒤를 볼 필요가 없음
        - ex2) || 연산 시 참이다 → 뒤를 볼 필요가 없음

### equals

- 문자열의 비교는 반드시 equals로 강제
    - `==` 는 주소 비교, `equals`는 값 비교
    - 힙 메모리에 “AAA” 하나만 존재하는 경우
    
    ```java
    String str1 = "AAA";
    String str2 = "AAA";
    
    System.out.println(str1==str2); // true
    ```
    
    - 힙 메모리에 저장된 주소 값이 다른 경우
        - new: 새로운 메모리를 만들라고 강제
    
    ```java
    String str3 = new String("BBB");
    String str4 = new String("BBB");
    
    System.out.println(str3==str4); // false
    str3.equals(str4); // true
    ```

### 간단한 메뉴 만들기 예제

내가 푼 코드

```java
Scanner sc = new Scanner(System.in);
		
		int balance = 0;
		int amount = 0;
		boolean exit = false;

		while(true) {
			System.out.println("1. 예금 | 2. 출금  | 3. 잔고 | 4. 종료");
			int input = Integer.parseInt(sc.nextLine());
			
			switch (input) {
				case 1:
					System.out.println("예금액: ");
					amount = Integer.parseInt(sc.nextLine());
					balance += amount;
					break;
				case 2:
					System.out.println("출금액: ");
					amount = Integer.parseInt(sc.nextLine());
					balance -= amount;
					break;
				case 3:
					System.out.printf("잔고: %d\n", balance);
					break;
				case 4:
					exit = true;
					break;
				default:
			}
			
			if (exit) break;
```
